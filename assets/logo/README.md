# Pandas Image Upscaler Logo

This directory contains the official logo for the Pandas Image Upscaler application.

## Design

The logo features a friendly panda face in Material Design style with:
- **Primary Color**: Material Green 500 (#4CAF50)
- **Style**: Flat, modern, and friendly
- **Features**: Cute panda with rosy cheeks and sparkling eyes

## Files

### Main Logo
- `panda-logo.png` - Main logo (512x512px) used in README

### Available in Static Assets
All logo sizes are available in `src/upscaler/static/logo/` for web use:
- `favicon.ico` - Multi-size favicon (16x16, 32x32, 48x48)
- `favicon-16x16.png` - Browser favicon
- `favicon-32x32.png` - Browser favicon
- `apple-touch-icon.png` - Apple devices (180x180)
- `panda-logo-{size}.png` - Various sizes: 48, 72, 96, 128, 144, 152, 192, 256, 512

## Usage

### In README
```markdown
<p align="center">
  <img src="assets/logo/panda-logo.png" alt="Pandas Image Upscaler Logo" width="256" height="256">
</p>
```

### In HTML
The logo is automatically included in the HTML template with all appropriate favicon and icon tags for:
- Desktop browsers
- Mobile browsers  
- iOS devices (Apple Touch Icon)
- Android devices (Web App Manifest)
- Progressive Web App support
